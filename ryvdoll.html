Ryvdoll

Newsletter Studio · France

A custom drag-and-drop email builder built from scratch for a French magnetic fashion doll brand — in one week.

01 — Problem

Brevo couldn't match the brand. So we replaced it.

Ryvdoll is a boutique French fashion doll brand with a precise visual identity. Brevo offered templates, not control. Every modification required workarounds. The brand's aesthetic couldn't survive a generic email builder. The solution wasn't to find a better template — it was to build the tool from scratch and own it entirely.

No design control

Brevo's templates imposed constraints that broke Ryvdoll's visual identity.

Modification hell

Every client request meant fighting the platform instead of executing the work.

Platform dependency

Data, templates, and send logic locked inside a third-party tool.

02 — Approach

Audit & Architecture

Mapped full email workflow. Defined block types, template logic, subscriber data structure.

Block Editor

Custom drag-and-drop in vanilla JS — text, image, button, divider, spacer. All reorderable, all editable inline.

Live Preview

HTML renders in real time. Save templates, reload, iterate. Full inline CSS compilation for email clients.

Subscriber + SMTP

Import/export, tag segmentation, direct send via Google Workspace SMTP.
