Ryvdoll

Newsletter Studio

A custom drag-and-drop email builder, built from scratch for a French magnetic fashion doll brand.

01 — Problem

Mailchimp was costing more than it was worth.

Ryvdoll needed full control over their email identity — custom blocks, branded layouts, subscriber segmentation. Off-the-shelf tools imposed limitations and growing costs with no flexibility for a boutique brand with a precise aesthetic vision.

No design control

Mailchimp templates couldn't match Ryvdoll's visual identity.

Escalating costs

Pricing scaled with subscribers, not with value delivered.

Zero ownership

Data, templates, and logic locked inside a third-party platform.

02 — Approach

Built to own. Built to last.

Audit & Architecture

Mapped Ryvdoll's full email workflow. Defined block types, template logic, and subscriber data structure before writing a single line of code.

Drag-and-Drop Engine

Built a custom block editor in vanilla JS — text, image, button, divider, and spacer blocks, all reorderable, all editable inline.

Template System & Preview

Live HTML preview rendering as you build. Save templates, reload them, iterate without friction.

Subscriber Management + SMTP

Full subscriber list with import/export, segmentation by tag, and direct send via Google Workspace SMTP — no third party in the critical path.
