Ryvdoll

Newsletter Studio · France

A custom drag-and-drop email builder built from scratch for a French magnetic fashion doll brand — in one week.

01 — Problem

Brevo couldn't match the brand. So we replaced it.

Ryvdoll is a boutique French fashion doll brand with a precise visual identity. Brevo offered templates, not control. Every modification required workarounds. The brand's aesthetic couldn't survive a generic email builder. The solution wasn't to find a better template — it was to build the tool from scratch and own it entirely.

No design control

Brevo's template system imposed constraints that broke Ryvdoll's visual identity at every turn.

Modification hell

Every client request meant fighting the platform instead of executing the work.

Platform dependency

Data, templates, and send logic locked inside a third-party tool with escalating costs.

02 — Approach

One week. Full ownership.

Audit & Architecture

Mapped Ryvdoll's email workflow end to end. Defined block types, template logic, and subscriber data structure before writing a single line of code.

Drag-and-Drop Block Editor

Built a custom editor in vanilla JS — text, image, button, divider, and spacer blocks, all reorderable, all editable inline. No framework. No overhead.

Live Preview & Template System

HTML preview renders in real time as you build. Save templates, reload them, iterate without friction. Full inline CSS compilation for email client compatibility.

Subscriber Management + SMTP

Full subscriber list with import/export and tag-based segmentation. Direct send via Google Workspace SMTP — no third party in the critical path.
