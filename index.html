We don't decorate the web. We engineer it.

Arcane Studio is a one-person AI-powered web development studio based in Tel Aviv. We operate two web agency brands — Mizra in Israel, Lancio in France — and build internal tools, automation pipelines, and custom platforms for clients who need things done fast and right. Every project starts with a real problem and ends with working software. No templates. No bloat. No subcontractors hiding behind a Figma file. Just direct, precise builds that ship. We use AI at every layer — to write code faster, to automate lead generation, to analyze documents, to send emails, to scrape data. Not as a gimmick. As infrastructure. We've built e-commerce platforms, newsletter studios, web agency back-offices, WhatsApp outreach pipelines, SEO crawlers, and SaaS tools — often in under a week. The Lab is where we push further: 30 mini-projects in 30 days, each one live, each one solving a real problem.
